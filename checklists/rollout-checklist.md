# Rollout Checklist – Remote Laptop Onboarding



\## VM Creation

\- \[ ] Create VirtualBox VM (Win10)

\- \[ ] Attach ISO

\- \[ ] Allocate 4 GB RAM

\- \[ ] Allocate 2 CPUs

\- \[ ] Create 50 GB VDI disk



\## Windows Installation

\- \[ ] Choose "Custom Install"

\- \[ ] Install to unallocated disk

\- \[ ] Complete Out-of-Box Experience (OOBE)





\## Initial Configuration

\- \[ ] Run Windows Update fully

\- \[ ] Rename PC to WIN10-EMPLOYEE-LAPTOP01

\- \[ ] Configure timezone \& region

\- \[ ] Take screenshots



\## Onboarding Simulation

\- \[ ] Attempt work account connection

\- \[ ] Capture Access Work/School screenshots

\- \[ ] Install Teams / OneDrive / Office tools

\- \[ ] Capture browser login screens

\- \[ ] Configure local policy (if Pro)





\## Security

\- \[ ] Open Windows Security

\- \[ ] Screenshot all panels

\- \[ ] Check Firewall running

\- \[ ] Check Windows Defender status



\## Evidence

\- \[ ] Save all screenshots in /Assets/Screenshots/



\## Documentation

\- \[ ] Update README.md

\- \[ ] Update KB document

\- \[ ] Update ticket log
